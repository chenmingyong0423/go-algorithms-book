# Go 数据结构与算法
大家好，我是陈明勇。

众所周知，数据结构与算法对程序员的重要性毋庸置疑，虽然对于很多程序员来说，它们似乎是可有可无的，因为在日常工作中很少直接用到。但其实，对于程序员来说，掌握数据结构与算法是修炼编程内功必不可少的知识。

本身我是一个热爱分享的人，因此在学习和巩固数据结构与算法知识的同时，我也想把自己对这方面的理解和思考分享给大家，希望能够帮助到大家。
